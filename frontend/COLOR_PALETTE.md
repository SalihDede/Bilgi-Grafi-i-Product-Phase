# Tasarım Anlayışı ve Renk Paleti

## Genel Yaklaşım

- Tek sayfa (single-page) tasarım
- Minimal ve editorial estetik
- Sade, temiz, içerik odaklı yapı

## Renk Paleti

| Renk | Hex | Kullanım |
|---|---|---|
| Krem Bej | `#E8E5DF` | Sayfa arka planı |
| Koyu Siyah | `#111111` | Ana başlıklar, bold metin |
| Kobalt Mavi | `#1B44D8` | Vurgu, italic metin, alt çizgi |
| Açık Gri | `#888888` | İkincil / küçük metinler |
